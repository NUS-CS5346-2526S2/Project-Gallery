Analysis logic

How associations are measured

The shortlist uses Spearman rank correlation because the featured variables are ordinal. The chart views then show row-normalized distributions so users can still see how categories are distributed inside each factor level.

Caveats

What still needs caution

Reverse causality is plausible, especially for academic stress, concentration difficulty, and sleep disruption. The question on night awakenings may also be interpreted differently by different respondents.

Benchmark layer

How the cross-country context should be read

The benchmark page compares country-level student-sleep records drawn from different studies. It adds context and scope, but it is not a fully harmonized pooled dataset and should stay descriptive.
